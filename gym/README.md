# Able Pro Inspired Admin Dashboard

This project scaffolds a Next.js 14 App Router dashboard styled in the spirit of Able Pro's validation layouts. It ships with Material UI, Redux Toolkit, React Hook Form + Zod, Axios, Recharts, and a suite of reusable components for cards, forms, tables, and charts.

## Scripts

- `npm install` – install dependencies
- `npm run dev` – start the development server on <http://localhost:3000>
- `npm run build` – create a production build
- `npm run start` – run the production server
- `npm run lint` – lint the project using ESLint

## Features

- Next.js App Router with TypeScript
- Material UI theming inspired by Able Pro
- Redux Toolkit store with example slices
- React Hook Form + Zod validation helpers
- Reusable layout primitives, steppers, and form cards
- Sample dashboard, forms, tables, and chart pages

## Getting Started

1. Install dependencies with `npm install`.
2. Launch the local dev server via `npm run dev`.
3. Open the app in your browser at `http://localhost:3000`.

Adjust styling, copy, and data sources to align with your own Able Pro license and requirements.
